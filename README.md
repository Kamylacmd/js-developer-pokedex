# Trilha JS Developer - Pokedex

💡 O que desenvolvi:

→ Consumo de dados da PokéAPI utilizando Fetch API
→ Renderização dinâmica dos Pokémon com JavaScript
→ Paginação da lista utilizando offset e limit
→ Modal/Card de detalhes para cada Pokémon
→ Exibição de informações adicionais como:
• Número da Pokédex
• Nome
• Tipos
• Imagem
• Experiência base
• Altura e peso
• Base Stats com barras de progresso

🛠 Tecnologias e conceitos praticados:

→ HTML5
→ CSS3
→ JavaScript (ES6+)
→ Consumo de APIs REST com Fetch
→ Promises e operações assíncronas
→ Manipulação do DOM
→ Eventos e interatividade
→ Organização de código em módulos

📚 Aprendizados do projeto:

Mais do que exibir dados de uma API, esse desafio me ajudou a compreender melhor o fluxo completo de uma aplicação front-end:

→ Como realizar requisições assíncronas
→ Como transformar respostas JSON em objetos da aplicação
→ Como atualizar a interface dinamicamente
→ Como estruturar funcionalidades de forma reutilizável e organizada

Também foi uma ótima oportunidade para praticar a leitura da documentação de uma API e implementar funcionalidades além das propostas inicialmente no desafio.
